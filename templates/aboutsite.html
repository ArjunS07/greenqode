Tech Stack

This website was written in Django, a robust framework meant for building full-stack web applications in Python. It is hosted on a Heroku dyno, and uses a PostgreSQL database to store relational data for communities. Third-party packages and libraries used include:

  • PyFPDF for facilitating the generation of PDFs
  • Firebase Storage for hosting the generated PDFs temporarily
  • Bulma for styling the site
  • Font Awesome for all the icons used in the site
