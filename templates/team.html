Hi, I'm Arjun Sharma

I'm an 11th grade student living in Gurgaon, India. I've been interested in all things tech since fifth grade, and have been coding since I was eleven. I like to write about my thoughts on technology on my personal blog.

I'm passionate about using technology to make a change in society. GreenQode is one of my efforts that attempt to realise this goal. The idea was born in a sustainability hackathon in March 2022. Since then, it's gone through multiple iterations aimed at increasing the usability and scalability of the web app.

My technical skills include:

  • Full-stack web development with Django (which is what this site is built with)
  • Data analysis and machine learning, using packages such as Tensorflow, Keras, and Turi Create
  • App development with Flutter, using Dart, as well as with UIKit, using Swift
  • Front-end web development using the languages of the web - HTMl, CSS, and JavaScript. I've worked with a number of front-end frameworks, including Bootstrap, Bulma (another component of this site), and Tailwind CSS.
  • Python and Java for scripting and other programming tasks

I'm always open to working on exciting and impactful projects. Feel free to email me at arjun.sharma07@outlook.com if you've got any ideas you'd like to work with me on - or for anything else. You can find my Github here.

Thanks for checking out GreenQode!
